# SEEDVALLEY

![Hyperlocal growing guide for Bengaluru](https://seeds.layogtima.com/images/seedvalley.png)

## What is SEEDVALLEY?

SEEDVALLEY is your hyperlocal guide to growing food in Bengaluru. It takes the guesswork out of gardening by providing season-specific advice tailored to our unique climate.

Think of it as a plant coach that actually knows what "water regularly" means for Bengaluru's distinct seasons, elevation, and urban growing conditions.

> "To plant a garden is to believe in tomorrow." — Audrey Hepburn

## Core Features

### Season-Specific Guidance

Recommendations that change with Bengaluru's unique seasons, not generic growing advice.

### Hyperlocal Focus

Details that matter: our elevation (920m), temperature patterns, and distinct wet/dry seasons.

### Container Gardening Solutions

Practical advice for balconies, terraces, and small urban spaces.

### Clear Instructions

Straightforward growing guides with specific watering, sunlight, and care information.

## Key Principles

### Grow What Thrives Here

Curated selection of plants that actually succeed in Bengaluru's climate.

### Plant By Season

Work with Bengaluru's seasonal patterns for the best results.

### Start With Confidence

Begin with easy-growing options and build your skills gradually.

## Tech Stack

Built with minimalist web technologies, embracing the constraint-driven approach of MONO:

- HTML5
- Tailwind CSS via CDN
- Vanilla JavaScript with modular architecture
- No backend - pure frontend for simplicity
- Space Mono typeface for consistent visual identity

## Getting Started

```bash
git clone https://github.com/layogtima/seedvalley.git
cd seedvalley
# Open index.html in your browser
# Or serve locally with:
python -m http.server
```

## Features

- **Seasonal Dashboard**: Updates based on current growing conditions
- **Plant Database**: Filter by difficulty, growing time, and season
- **Dark Mode**: Easy on the eyes for nighttime planning
- **Multilingual**: English and Kannada support
- **Interactive Onboarding**: Quick introduction to key features
